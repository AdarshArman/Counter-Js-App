# Counter Js App

A simple counter application built using **HTML, CSS, and JavaScript**.

---

## Features

- Increase counter
- Decrease counter
- Change the bars color on Increase or Decrease 
- Simple and clean UI

---

## Technologies Used

- HTML
- CSS
- JavaScript
